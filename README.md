   Overview
CineMax is a modern and visually immersive web application designed for movie lovers. It provides a cinematic platform where users can explore movies, watch trailers, and discover new releases — all within a sleek, dark-themed interface inspired by real streaming platforms.

   Features

  Browse movies and upcoming releases
   Watch trailers directly from the platform
   Search functionality for quick movie discovery
  Newsletter subscription section
  Ticket booking UI (frontend concept)
  Fully responsive design for all devices
  Dark mode UI with cinematic styling

   Technologies Used

Frontend: HTML, CSS, JavaScript, Bootstrap
Backend: Python (Django)
Version Control: Git & GitHub

   What I Learned

Creating cinematic and engaging UI designs
Building responsive layouts with Bootstrap
Enhancing interactivity using JavaScript
Structuring backend logic with Django
Designing user-friendly navigation and content flow

   Purpose of the Project
This project was built to practice full-stack web development while focusing on UI/UX design inspired by real-world streaming platforms. It also helped me improve my ability to combine aesthetics with functionality.
